🐡 About Is This Phishy?

A free Australian scam detection tool. Paste any suspicious message — SMS, email, WhatsApp — and get an instant plain-English analysis of whether it's a scam.

Built specifically for Australians. Covers local scam types including AusPost, CommBank, NAB, ATO, Centrelink, Linkt, and more.

🔒 Privacy

✓Message text is never stored — analysed instantly and discarded
✓No account, no email, no tracking, no ads
✓Your check history lives only in your browser's local storage
✓Stats only count verdicts, never message content

📣 Report Scams

If you've received a scam, report it to help protect other Australians:

🏛️Scamwatch (ACCC) — the main Australian scam reporting body
🏦AFCA — if your bank didn't help after a financial loss
🔐ASD Cyber.gov.au — for cybercrime and hacking incidents

⚙️ How detection works

1.Feature extraction — URLs, phone numbers, brand mentions, domain mismatches
2.Rule engine — 14+ structural rules scoring individual signals
3.Pattern engine — 50 compiled regex patterns across 10 scam categories
4.Scam classifier — maps signals to a human-readable scam type
